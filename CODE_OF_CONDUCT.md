# Contributor Covenant Code of Conduct

## Our Pledge
We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards
Examples of behavior that contributes to a positive environment:
- Being respectful and constructive
- Giving and receiving feedback gracefully
- Focusing on what is best for the community and users

Examples of unacceptable behavior:
- Trolling, insults, or derogatory comments
- Harassment in public or private channels
- Publishing private information without explicit permission

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing standards.

## Scope
This Code of Conduct applies within all project spaces and public spaces when representing the project.

## Enforcement
To report violations, contact the maintainers via repository issue with label `conduct` or private maintainer email.
